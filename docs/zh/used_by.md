    ---
    一个基于 PyTorch 的推荐系统框架，强调**标准化训练/评测流程**与**可复现实验**。

    亮点：

    - 覆盖 30+ 主流推荐模型（排序 / 召回 / 多任务 / 生成式）

    - 统一的数据加载、训练、评估流程
    - 支持导出 ONNX，便于部署
